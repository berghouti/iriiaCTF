What is a CTF?

Capture The Flag — the hacker's sport. Learn everything you need to get started.

// Definition

🏴 Capture The Flag

A CTF (Capture The Flag) is a cybersecurity competition where participants solve security puzzles to find hidden strings called flags. Each challenge tests a specific hacking skill — web exploitation, cryptography, forensics, reverse engineering, and more.

// Flag Format

⚑ What Does a Flag Look Like?

Every flag on this platform follows the format below. Once you find it, paste it into the submission box to earn points.

iriiaCTF{s0m3_s3cr3t_h3r3}

Flags can be hidden in page source, inside files, encoded in text, printed by scripts, and more.

// Categories

📂 Challenge Types

Web: Find bugs in websites — hidden comments, cookie tampering, SQL injection.

Crypto: Decode encrypted messages — Caesar ciphers, Base64, XOR encryption.

Forensics: Investigate files — extract hidden data, read metadata, use strings.

Linux: Command-line mastery — find, grep, chmod, pipes, and more.

// Live Walkthrough — Solve Your First CTF

EXAMPLE CHALLENGE — CAESAR CIPHER

You receive this ciphertext: luullNFWC{fkliwhu_flskhu_lv_ixq}

The challenge says: "Caesar used a shift of 3". Find the flag.

1

Read the Challenge Carefully

The challenge tells us it's a Caesar cipher with a shift of 3. That means every letter was shifted forward by 3. We need to shift backward by 3 to decode it.

2

Choose Your Tool

Go to gchq.github.io/CyberChef, search for "ROT13" in the operations panel, and set the amount to 3. Or use Python:

3

Apply the Decode (Python)

Open a terminal and run this snippet:

cipher = "luullNFWC{fkliwhu_flskhu_lv_ixq}" result = "" for c in cipher: if c.isalpha(): base = ord('A') if c.isupper() else ord('a') result += chr((ord(c) - base - 3) % 26 + base) else: result += c print(result) # → iriiaCTF{cipher_cipher_is_fun}
4

Submit the Flag

Copy the decoded flag iriiaCTF{cipher_cipher_is_fun}, paste it into the challenge submission box, and click Submit. You earn points! 🎉
